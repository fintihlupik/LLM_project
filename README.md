iniciar proyecto:
Crear el entorno virtual con venv:

Linux/Mac
```textplain
python3.10 -m venv .venv
```
Windows
```textplain
python -m venv .venv
```

Inicio el entorno virtual

Linux/Mac
```textplain
source .venv/bin/activate
```
Windows
```textplain
.venv\Scripts\activate
```

Instalar dependencias
```textplain
pip install -r requirements.txt
```
